Pin Connections:
  - Water Sensor: A0
  - DHT11: Digital Pin 7
  - LCD: Pins 12, 13, 30, 32, 34, 36
  - RTC: I2C (SDA/SCL - Pins 20/21)
  - Stepper Motor: Pins 8, 9, 10, 11
  - Fan Motor: Pin 5 
  - Yellow LED: Pin 22
  - Green LED: Pin 24
  - Blue LED: Pin 26
  - Red LED: Pin 28
  - Start Button: Pin 2
  - Stop Button: Pin 3
  - Reset Button: Pin 19
